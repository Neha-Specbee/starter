---
title: "Google Tag Manager"
description: "It is just a couple of steps to setup the FullContact tag with Google Tag Manager."
---

In Google Tag Manager - Create a new tag, and under Tag Configuration, select Custom HTML. Paste your generated FullContact Tag and give it an appropriate name.

To help work through this configuration - Click the following image to expand a visual walkthrough for adding the FullContact tag inside of the Google Tag Manager user interface.

The tag is now set up to use with Google Tag Manager!
